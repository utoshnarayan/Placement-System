# Placement Record Management System

A modern, responsive frontend for managing student placement records, built with vanilla HTML, CSS, and JavaScript.

## Features

- **Public Views**
  - Landing page with key statistics
  - Placement records listing with search, filter, and pagination
  - Company directory
  - Analytics and visualizations

- **Admin Features**
  - Secure admin login (mock authentication)
  - CRUD operations for placement records
  - Bulk import/export of data (CSV/JSON)
  - Activity logging
  - Dashboard with overview metrics
